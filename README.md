# Food-Nutrition-Tracking-Guide
Food Nutrition Tracker Guide is a dynamic web-based application designed to help users monitor and manage their daily nutritional intake. The system allows users to add, view, and delete food records while automatically calculating total calories, protein, carbohydrates, and fats consumed.
This project focuses on promoting healthy lifestyle habits by giving users a clear overview of their dietary consumption in a visually attractive and interactive way. It includes secure login functionality, animated UI effects, and real-time nutritional summary calculations.
The application is user-friendly, responsive, and built with a modern animated interface to enhance user experience.


**Features:**
User authentication system (Login & Logout)
Add food details (Food Name, Calories, Protein, Carbs, Fats)
Delete food records
View all food entries in a structured table format
Automatic calculation of total Calories
Automatic calculation of total Protein
Automatic calculation of total Carbohydrates
Automatic calculation of total Fats
Animated and interactive user interface
Gradient background and glassmorphism design
Smooth hover effects and transitions
Responsive layout design
Secure session management
Database integration using MySQL
Real-time nutritional summary display
